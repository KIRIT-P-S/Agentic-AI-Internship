# 15 Days Agentic AI Internship

## Overview

This repository contains the projects, assignments, experiments, and learning outcomes completed during a 15-day Agentic AI Internship. The internship focused on building intelligent AI agents, integrating Large Language Models (LLMs), tool usage, automation workflows, and developing real-world AI-powered applications.

## Objectives

* Understand Agentic AI concepts and architectures
* Build AI agents capable of reasoning and tool usage
* Integrate modern LLMs into applications
* Develop multimodal and task-oriented AI systems
* Explore AI workflow automation and orchestration
* Deploy AI-powered applications

## Installation

Clone the repository:

```bash
git clone https://github.com/KIRIT-P-S/Agentic-AI-Internship.git
```

Navigate to the project directory:

```bash
cd Agentic-AI-Internship
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run individual projects from their respective day folders:

```bash
python app.py
```

or

```bash
streamlit run app.py
```
